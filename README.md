# Multi-Drone Flight Simulator with PLATEAU 3D City Data

A multi-drone flight simulator built with Unity and PLATEAU SDK for Unity, featuring two-phase A* path planning for urban drone navigation.

This simulator was developed as part of the following research paper:
> "Multi-Drone Flight Simulation in Real Urban Environments Using PLATEAU 3D City Data and Two-Phase A* Path Planning" (Accepted at SNPD 2026)

---

## Repository Structure

| Folder | Description |
|---|---|
| `DroneSimulator/` | Unity project (main simulator) |
| `LogVisualizer/` | Flight log visualization tool (Python) |
| `TCPClient/` | Drone control client application (Python) |

---

## Requirements

### DroneSimulator
- Unity 2022.3 LTS
- PLATEAU SDK for Unity v4.2.0

### LogVisualizer / TCPClient
- Python 3.8+
- matplotlib
- tkinter (included in standard Python distributions)

---

## Features

- Simultaneous control of up to 8 drones via TCP/IP
- Two-phase A* path planning for urban obstacle avoidance
- Six-layer weather simulation (wind, gusts, updrafts, downdrafts, rain, fog)
- Four drone types with different flight characteristics
- Physics-based flight dynamics with PID altitude control
- Inter-drone repulsion and downwash effects
- Flight log recording and visualization

## Getting Started

### 1. DroneSimulator (Unity)

1. Download PLATEAU 3D city data for your target city from the [PLATEAU official site](https://www.mlit.go.jp/plateau/)
2. Open `DroneSimulator/` in Unity 2022.3 LTS
3. Import the downloaded PLATEAU data using PLATEAU SDK for Unity
4. Open `TitleScene` and press Play
5. Select **Keyboard Mode** for single-drone manual operation, or **TCP Mode** for multi-drone control via the Python client

> **Note:** In TCP Mode, enable **Run In Background** in Unity (Edit → Project Settings → Player → Resolution and Presentation) to prevent the server from pausing when Unity loses focus.

### 2. TCPClient (Python)

```bash
cd TCPClient
python client.py
```

Enter the IP address and port, add drones, and start autonomous flight.

### 3. LogVisualizer (Python)

```bash
cd LogVisualizer
python visualizer.py
```

Load CSV flight logs generated by the simulator to visualize time-series data and 3D flight paths.
